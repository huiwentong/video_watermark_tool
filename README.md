# VibeWatermark

一款桌面视频水印工具，支持实时预览、多种水印样式，基于 Python、PySide6 和 ffmpeg 构建。

---

## 功能特性

- **可视化预览面板** — 每项参数调整后立即看到效果
- **图片水印** — 支持 PNG / JPG / BMP / TIFF / WebP 格式
- **文字水印** — 自定义文字内容与颜色，自动检测中文字体（微软雅黑）
- **每项水印独立控制：**
  - 透明度（0%–100%）
  - 缩放（5%–500%）
  - 旋转角度（-180° 到 180°）
  - 9 个预设位置 + 自定义 X/Y 坐标
  - 平铺模式：无平铺 / 动态平铺 / 填充画面
- **多水印叠加** — 同时添加任意数量的图片水印和文字水印
- **实时预览** — 拖动滑块即时更新画面效果
- **ffmpeg 导出** — 后台调用 ffmpeg 滤镜图渲染输出
- **深色专业界面** — Katana 风格暗色主题

---

## 环境要求

| 依赖 | 用途 |
|---|---|
| Python 3.10+ | 运行环境 |
| PySide6 | Qt 图形界面框架 |
| Pillow | 预览合成与预渲染 |
| opencv-python-headless | 视频帧提取 |
| ffmpeg（4.x 以上，需在系统 PATH 中） | 视频编码与滤镜图执行 |

### 预览面板（左侧）

1. 点击 **浏览** 选择视频文件（支持 MP4 / AVI / MOV / MKV 等格式）
2. 视频首帧加载后显示在预览区域
3. 底部信息栏显示视频尺寸

### 水印控制台（右侧）

| 区域 | 说明 |
|---|---|
| **标题栏** | "Watermark Console" 面板标题 |
| **列表** | 显示所有已添加的水印；点击选中 |
| **+ Add Watermark** | 弹出对话框添加图片或文字水印 |
| **− Delete** | 删除当前选中的水印 |
| **属性编辑** | 调整选中水印的透明度、缩放、旋转、位置、平铺 |

---

## 添加水印

点击 **+ Add Watermark** 打开添加对话框：

### 图片水印
1. 选择 **Image Watermark**
2. 点击 **Browse…** 选择图片文件
3. 点击 **Add**
4. 水印出现在列表中并渲染到预览画面

### 文字水印
1. 选择 **Text Watermark**
2. 输入水印文字和十六进制颜色值（如 `#FFFFFF`）
3. 点击 **Add**
4. 文字出现在列表中并渲染到预览画面

---

## 调整水印属性

在列表中选中水印后，使用属性编辑区域：

| 控制项 | 范围 | 说明 |
|---|---|---|
| **Opacity（透明度）** | 0% – 100% | 水印的不透明度 |
| **Scale（缩放）** | 5% – 500% | 相对于原始尺寸的比例 |
| **Rotation（旋转）** | –180° – 180° | 顺时针旋转（预览与导出方向一致） |
| **Position → Preset（位置预设）** | 9 种 + 自定义 | 快速定位到四角、四边或中心；或手动输入 X/Y |
| **Tiling → Mode（平铺模式）** | None / Tile / Fill | None = 单次；Tile = 重复铺满；Fill = 拉伸铺满 |

> 任何调整都会**立即更新预览**画面。

---

## 导出视频

1. 确保至少添加了一个水印
2. 点击 **Export Video**
3. 选择输出文件夹
4. 工具在后台调用 ffmpeg 渲染，生成 `[原文件名]_watermarked.mp4`
5. 进度条显示导出状态，完成后弹出提示框

若导出失败，弹窗会显示 ffmpeg 的具体错误信息，帮助定位问题。

## 常见问题

| 问题 | 可能原因 | 解决方法 |
|---|---|---|
| 导出失败 | ffmpeg 未添加到系统 PATH | 安装 ffmpeg（4.x 以上）并加入 PATH |
| 导出失败 — drawtext 错误 | 字体文件找不到或不兼容 | 工具会自动检测系统中的微软雅黑字体 |
| 导出失败 — 滤镜图错误 | 混合文字+图片水印时流索引不对 | 当前版本已自动预渲染处理 |
| 中文无法显示 | 系统中文字体缺失 | 确保 `C:\Windows\Fonts` 下有中文字体 |
| 预览显示空白 | 视频编码不被 OpenCV 支持 | 建议使用 H.264 编码的 MP4 文件 |
